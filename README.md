## get running

```bash
# 1. clone the repo
git clone https://github.com/joseph-beck/joseph-beck.com

# 2. install pnpm if missing
npm i -g pnpm

# 3. install packages
pnpm install

# 4. run your app
pnpm run dev

# 5. connect on localhost:5173!
```
